OnePlus Ace5 Ultra 内核构建项目

2025-09-27

OnePlus Ace5 Ultra 内核构建项目

警告：系统版本为610的用户请勿使用本项目，根据用户反馈610系统刷入内核后会无限重启。

并且目前一加官方尚未开源最新的610内核源码（官方仓库），我们无法进行适配工作。

项目概述

这是一个基于 GitHub Actions 的自动化内核构建工作流，专为 OnePlus Ace5 Ultra 设备定制。项目支持多种 Root 管理器和性能优化功能，提供稳定可靠的内核构建解决方案。

核心功能特性

Root 管理器支持

  • SukiSU Ultra - 增强版 SukiSU 权限管理框架（默认选择）
  • KernelSU Next - 下一代内核级权限管理系统
  • KernelSU - 原版内核级权限管理
  • KPM 模块支持（仅对 SukiSU-Ultra 生效，谨慎开启，可能导致无法开机）

文件系统优化

  • SUSFS 文件系统 - 支持魔术挂载、路径重定向和符号链接欺骗功能

压缩算法选项

  • lz4 1.10.0 + zstd 1.5.7 - 更新的压缩算法支持（默认选择）
  • lz4kd - 高效内存压缩算法（可选启用）
  • 全部安装 - 同时启用两种压缩优化

网络优化

  • BBR 拥塞控制算法 - 可选仅加入算法或设为默认 TCP 拥塞控制
  • 支持多种拥塞控制算法：CUBIC、VEGAS、NV、WESTWOOD、HTCP、BRUTAL

I/O 调度器

  • ADIOS IO 调度器 - 结合截止时间优先调度与自适应延迟控制算法，降低 I/O 延迟

系统优化

  • Re-Kernel 支持 - 用于与 Freezer、NoActive 等软件配合使用，提升冻结后台能力
  • 风驰调度器 - 官方原版风驰（scx）调度器

安全保护机制

  • Baseband-guard 防格机 - 内核层面阻止对关键分区/设备节点的非法写入（默认开启）

自定义选项

  • 内核后缀 - 自定义内核版本标识
  • 内核构建日期 - 修改内核版本信息中的构建时间戳

快速开始

环境要求

  • GitHub 账户
  • 已 Fork 本项目到个人仓库

构建步骤

  1. 访问 GitHub Actions
    • 进入您 Fork 后的仓库页面
    • 点击顶部导航栏的 “Actions” 选项卡
  2. 选择工作流
    • 在左侧边栏选择 “Oneplus_Ace5_Ultra_GKI” 工作流
  3. 配置构建参数
    • 点击 “Run workflow” 按钮
    • 根据需求配置以下参数：
      • KernelSU分支：选择 Root 管理器类型（默认：SukiSU-Ultra）
      • 是否开启KPM：仅对 SukiSU-Ultra 生效（默认：false）
      • lz4 补丁选项：选择压缩算法优化（默认：安装lz4&zstd补丁）
      • 基带保护：启用内核级基带保护（默认：true）
      • BBR算法：网络拥塞控制优化（默认：关闭）
      • ADIOS调度器及Re-Kernel：I/O 和系统优化（默认：均不安装）
      • 内核后缀：自定义内核标识
      • 内核构建日期：修改版本信息中的时间戳
  4. 启动构建过程
    • 点击 “Run workflow” 开始自动化构建
    • 构建过程通常需要 15-30 分钟

构建输出

构建完成后，在工作流的 Artifacts 区域可以下载 AnyKernel3 刷机包，文件名格式为：Kernel_[类型]_v[版本]_[时间戳]_[特性标志].zip

技术特性说明

编译优化

  • 使用 LLVM Clang 18 工具链编译
  • 启用 O2 编译优化
  • 集成 ccache 编译缓存，提升构建速度

兼容性支持

  • Android 版本：Android 15
  • 内核版本：6.6

支持与反馈

提出 BUG 反馈之前，请检查是否为管理器自身/上游问题，因为部分管理器自身 BUG 较多

如遇技术问题，请按以下步骤排查：

  • 确认设备型号为 OnePlus Ace5 Ultra
  • 验证 Android 版本与内核版本匹配
  • 刷机前务必备份重要数据
  • 确保电池电量充足（建议 >50%）
  • 若要反馈问题，需给出完整日志

法律声明与免责条款

使用条款

  • 本内核仅供技术学习和研究使用
  • 用户需自行承担刷机风险
  • 作者不对因使用本内核导致的设备问题负责
  • 若有任何倒卖行为，则替开发者挡灾挡难

项目支持

如果您觉得这个项目对您有帮助，请考虑给予 Star ⭐ 支持项目发展。

对于技术问题和功能建议，欢迎通过 GitHub Issues 反馈。

注意：刷机有风险，操作需谨慎。请在充分了解相关技术知识后进行刷机操作。
